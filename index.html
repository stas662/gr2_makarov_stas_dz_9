Цель курса
Совершенствование общепрофессиональных компетенций педагогов в области подготовки обучающихся 8-11 классов образовательных учреждений России к сдаче ОГЭ и ЕГЭ по немецкому языку
Курс подойдет для педагогов общеобразовательных организаций по немецкому языку
Форма обучения: дистанционная
Трудоемкость программы: 72 часа

Программа
курса
  • 1.1 Нормативные документы, структура и содержание ОГЭ по немецкому языку
  • 1.2 Нормативные документы, структура и содержание ЕГЭ по немецкому языку
  • 2.1 Основные принципы составления тестовых заданий различных типов
  • 2.2 Тестирование коммуникативных компетенций (чтение)
  • 2.3 Тестирование коммуникативных компетенций (письменная речь)
  • 2.4 Тестирование коммуникативных компетенций (аудирование)
  • 2.5 Тестирование коммуникативных компетенций (устная речь)
  • 2.6 Тестирование коммуникативных компетенций (лексика и грамматика)
